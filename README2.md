# Command History Logger

A Python script that records and displays command history.

## Description
This project allows users to save commands and view them later, similar to a terminal history feature.

## Features
- Save commands
- View command history
- Persistent storage using file

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python history_logger.py

4. Choose options from menu

## Example

1. Add Command  
2. View History  

Output:
ls -la  
python app.py
